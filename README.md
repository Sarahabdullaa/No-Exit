# No-Exit

NO EXIT is a psychological mystery adventure game where the player is trapped inside a surreal dream world. The game takes place in a hallway connected to multiple rooms, each representing a memory from the character’s past.

As the player explores rooms, solves puzzles, and collects important items, they slowly uncover the truth behind a tragic incident involving the character’s twin brother. The atmosphere becomes darker and more unsettling as the story progresses, leading to a final emotional revelation.

The main objective is to solve puzzles, collect memory pieces, unlock rooms, and ultimately escape the dream.
